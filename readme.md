## Galeria de Fotos de Animais
Este projeto é uma aplicação web interativa desenvolvida como exercício de programação para Aceleradora Ágil. A aplicação permite visualizar, filtrar e pesquisar imagens de animais através de uma interface responsiva e dinâmica.

## Funcionalidades
De acordo com os requisitos solicitados:  
Busca em Tempo Real: O sistema filtra as fotos instantaneamente conforme o usuário digita o nome do animal.  
Galeria Dinâmica: Exibição de um conjunto de dados estáticos com 10 fotos de animais e seus respectivos títulos.  
Layout em Grid: Organização das imagens em um sistema de grid moderno e organizado.  
Interatividade: Efeito visual de aumento de escala (zoom) e sombra ao passar o mouse sobre as fotos para melhorar a experiência do usuário.  
Feedback de Pesquisa: Exibição da mensagem "Nenhuma foto encontrada" caso nenhum critério de busca coincida com os títulos.  
Responsividade: Interface projetada para se ajustar a diferentes tamanhos de tela e orientações.  

## Tecnologias Utilizadas
Para atender aos requisitos técnicos:  
HTML5: Estruturação semântica da página.
CSS3: Estilização, layout Grid, Flexbox e efeitos de transição.
JavaScript (ES6+): Lógica de filtragem, normalização de caracteres (ignora acentos) e manipulação do DOM.
Font Awesome: Ícones utilizados na barra de busca. 

##  Estrutura de Arquivos

index.html: Contém o cabeçalho, a barra de busca, o container da galeria e o rodapé.


style.css: Define as cores (laranja e branco), fontes e o comportamento visual do grid.

script.js: Gerencia a lógica de busca e renderização dos cards de fotos.
